IDS validator 101
==========

In this IDS validator training, we introduce you to the basic concepts and features of
the IDS validator. You will need some basic familiarity with Python. For a refresher, see
the `Python tutorial <https://docs.python.org/3/tutorial/>`_. We also assume
some basic knowledge of the ITER IMAS infrastructure, see
https://imas.iter.org. You can find any info for IDS validator you need in the `IDS validator documentation<>`_.
